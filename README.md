## À propos

Ce portfolio interactif est une vitrine de mes compétences, projets et expériences en tant que Développeuse Web Full Stack. J'y expose ma maîtrise des technologies front-end et back-end (Java, PHP, Laravel, JavaScript, TypeScript, React), ainsi que ma capacité à concevoir des architectures logicielles robustes et des interfaces modernes.

## Fonctionnalités Clés

- **Design "Cosmique & Girly"** : Une interface moderne (Glassmorphism, animations fluides) avec une palette de couleurs personnalisée et affirmée.
- **Dark / Light Mode** : Bascule dynamique entre un thème sombre profond et un thème jour lumineux.
- **Intégration API GitHub (Live)** : Les statistiques de mes projets (Stars et Forks) sont récupérées dynamiquement via l'API publique de GitHub.
- **Graphe de Contributions** : Affichage en direct de mon activité de code via un widget GitHub Contributions.
- **Système de Filtrage Avancé** : Sur la page des archives, les projets peuvent être filtrés instantanément par technologie (Java, PHP, Laravel, JS, TS).
- **Internationalisation (i18n)** : Architecture prête pour le multilinguisme avec un système de traduction JSON personnalisé.
- **Responsive Design** : Entièrement optimisé pour ordinateurs, tablettes et mobiles grâce à Tailwind CSS.

## Technologies Utilisées

- **Frontend** : HTML5 sémantique, CSS3 (Variables, Animations), JavaScript (Vanilla ES6+), Tailwind CSS
- **Data Fetching** : Fetch API (Asynchrone)
- **Déploiement & CI/CD** : GitHub Actions, GitHub Pages
- **Design & UI** : FontAwesome 6, Google Fonts (Space Mono)
